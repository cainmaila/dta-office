將辦公桌+人物組合視為完整的工作站單元，創造更真實的辦公環境體驗。

### 🏢 兩種場景模式

#### 1. 👤 純角色模式 (npc.png)
```typescript

```
**特點:**
- 獨立的人物角色
- 適合走廊、會議區域
- 支援移動動畫
- 互動靈活

#### 2. 🏢👤 工作站模式 (npc-in.png)
```typescript
// 完整的辦公桌+人物場景

```
**特點:**
- 辦公桌+人物組合
- 真實的工作環境
