## 🧪 測試方法

### 基本功能測試
1. **點擊任意NPC** → 應該顯示對話氣泡
2. **觀察氣泡外觀** → 白底黑字圓角方形
3. **檢查尾巴指向** → 三角形指向NPC
4. **等待4秒** → 氣泡自動消失
5. **快速點擊多個NPC** → 只顯示最新的氣泡

### 視覺效果測試
- **氣泡位置**: 應該在NPC頭頂上方
- **邊界處理**: 氣泡不會超出螢幕邊界
- **文字換行**: 長文字自動換行顯示
- **動畫流暢**: 彈出和消失動畫自然

## 🐛 可能的問題
1. **氣泡未顯示** → 檢查控制台是否有JavaScript錯誤
2. **位置不正確** → 可能需要調整氣泡定位邏輯
3. **文字顯示異常** → 檢查字體和樣式設定
4. **動畫不流暢** → 檢查Phaser tween設定
